# GitHub Pages (Documentation & Portfolio)

This `docs` folder serves as the foundation for my GitHub Pages, an integral part of my professional online presence and portfolio. It contains all the necessary files and configurations to publish my portfolio website.

## Contents

-   **Website Content**: Markdown files, HTML templates, and assets that form the pages of my portfolio website.
-   **Configuration**: Files like `_config.yml` that configure the Jekyll site generation for GitHub Pages.
-   **Assets**: Images, stylesheets, and JavaScript files used across the website.
-   **Supporting Documentation**: Any additional files related to the setup, maintenance, or content creation for my GitHub Pages.

## Purpose

The primary purpose of this `docs` folder is to host and manage my personal portfolio website through GitHub Pages. It's a key component of my workforce readiness efforts, allowing me to showcase my projects, skills, and learning journey to potential employers and the wider tech community.
